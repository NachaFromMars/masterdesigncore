# masterdesigncore — Production-grade UI/UX for web and mobile

> A design-system skill that turns vague "make it look good" requests into disciplined, rule-driven interfaces. Covers Next.js/React (web) and React Native/Expo (mobile) with 30 golden rules.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
masterdesigncore is a UI/UX mastery skill that encodes 30 golden design rules into a repeatable process for both web and mobile platforms. It enforces best practices for visual hierarchy, color, typography, spacing, animation, and component design. On web it works with Tailwind CSS and shadcn/ui; on mobile it uses NativeWind for React Native/Expo. Use it to design new apps, review existing interfaces, or systematically improve a weak UI.

## Features
- **30 golden rules** covering hierarchy, color, typography, spacing, animation, components
- **Visual hierarchy** — max 3 focal points, primary CTA most prominent
- **Color discipline** — 60-30-10 rule, no pure black (#000), WCAG 4.5:1 contrast
- **Web stack** — Tailwind CSS + shadcn/ui
- **Mobile stack** — NativeWind (React Native/Expo)
- **Checklist-driven** — run final checklist before shipping

## Usage / Quick Start
1. Read SKILL.md → 2. Choose platform (web or mobile) → 3. Follow design system → 4. Apply 30 rules → 5. Run checklist.
